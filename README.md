🛍️ Drip Store

Projeto de e-commerce desenvolvido com React e Vite.

🚀 Tecnologias utilizadas
React
Vite
Tailwind CSS
React Router
📦 Como rodar o projeto
npm install
npm run dev
🏗️ Build para produção
npm run build
🌐 Deploy

O projeto está hospedado na Vercel.
